Read more about the module at: [Gpresult Like Tool For Intune](https://techcommunity.microsoft.com/blog/coreinfrastructureandsecurityblog/gpresult-like-tool-for-intune/4437008)

Install the module from the PowerShell Gallery for the current user using the following command:

```powershell  
Install-Module -Name IntuneDebug -Scope CurrentUser
```

or for all users on the computer using the following command (requires admin rights):

```powershell
Install-Module -Name IntuneDebug -Scope AllUsers
```
